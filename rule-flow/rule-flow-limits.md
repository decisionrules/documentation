  one rule flow. These limits vary across the different tariffs.
---

# Rule Flow Limits

### SaaS

| Plan     | Node Limit |
| -------- | ---------- |
| Tiny     | 0          |
| Small    | 10         |
| Medium   | 25         |
| Large    | 30         |
| Tailored | 50         |

### On-Premise

Node limit for on-premise plan is 30.

### Private Cloud

&#x20;Node limit for private cloud is also 30.
