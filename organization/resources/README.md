# Resources

This section covers the three foundational building blocks of your organization. By themselves, these resources are inactive. To grant access and enable collaboration, you must combine one of each to create a Policy.

* [Teams](teams.md): Learn how to organize existing members into functional groups for streamlined management.
* [Space Roles](space-roles.md): Learn how to define permission levels (e.g., Editor, Reader) and control what actions users can perform.
* [Spaces](spaces.md): Learn how to create and manage the actual project folders where your rules and work are stored.
