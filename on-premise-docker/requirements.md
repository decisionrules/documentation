# Requirements

### Software requirements

| Software       | OS                          |
| -------------- | --------------------------- |
| Docker Desktop | Windows 8.1 (WSL 2 enabled) |
|                | Windows 10 (WSL 2 enabled)  |

| ---------- | ---------------- |
